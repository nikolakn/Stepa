# Stepa
Hex game

##Build Dependencies##

- [GLEW](http://glew.sourceforge.net/)
- [SDL2](http://www.libsdl.org/)
- [ASSIMP](http://assimp.sourceforge.net/)
- [FreeType](http://www.freetype.org/)
- [FreeImage](http://freeimage.sourceforge.net/download.html)
- [qmake](http://www.qt.io/)

build:
qmake;
make
